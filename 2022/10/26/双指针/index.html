简介 双指针操作数组

介绍

双指针能通过一个快指针和慢指针在一个for循环下完成两个for循环的工作，能减少程序的时间和空间，对于删除，查找，排序元素的优化有很大的作用。
定义慢快指针：
1.快指针：寻找新数组的元素 ，新数组就是不含有目标元素的数组
2.慢指针：指向更新 新数组下标的位置

双指针查找元素

双指针每次循环都从两边开始，减少运行时间

public static int search(int[] nums , int target) {
   int length = nums.length;
   for(int i=0;i<=length-1;i++) {
       if(nums[i]==target)
           return i;
       if(nums[length-1-i]==target) //数组的开始和结尾一起查找
           return length-i-1;
   }
   return -1;
}

双指针移除元素

数组中的删除是用覆盖完成的，双指针能在一个for循环中完成操作

/**
* 相向双指针方法，确保了移动最少元素
* 时间复杂度：O(n)
* 空间复杂度：O(1)
*/
public static int removeElement(int[] nums , int target) {//返回数组的删除后的长度，数组发生了改变
        // 快慢指针
    int slowIndex = 0;
    for (int fastIndex = 0; fastIndex < nums.length; fastIndex++) {
        if (nums[fastIndex] != target) {//相等时，下次循环fastIndex会比slowIndex大
            nums[slowIndex] = nums[fastIndex];//利用一直赋值的方法
            slowIndex++;
        }
    }
    return slowIndex;
}
