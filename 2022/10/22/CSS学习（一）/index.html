简介 CSS学习

  • 介绍
    CSS主要有两部分组成：选择器以及一条或几条声明，放在style标签中,style标签放在head标枪中,可以参考CSS选择器

    基础选择器

  • 标签选择器
    <head>
<style>
  /*标签选择器不能差异化设置*/
  p {  /*p和{之间要有一个空格*/
      /*将p标签的属性改为设的值*/
      color: blue; /*：和属性值之间要有一个空格*/
      font-size: 12px;
  }
</head>
</style>
  • 类选择器
    <head>
<style>
  /*目标标签添加class="red"*/
  .red { /*类前面加一个.*/
    color: red;
    font-size: 12px;
  }
</style>
  • id选择器
    <head>
  <style>
      #pink { /*前面加一个# id是唯一的 只能被调用一次*/
        color: pink;
        font-size: 12px;
      }
  </style>
<head>
<body>
<table>
    <p id = "pink">123</p>/*添加类*/
</table>
</body>
  • 通配符选择器
    <head>
<style>
  * { /*将所有的标签都改属性值*/
    color: pink;
    font-size: 12px;
  }
</style>
<head>

字体属性

CSS font用于更改字体

  • 字体系列

    font-family: '宋体',serif;/*font-family改字体系列 可写多个字体*/

  • 字体大小

    font-size: 20px;/*font-size改字体大小 别忘了px*/

  • 字体粗细

    font-wight: 700;/*font-weight改字体粗细  700加粗 400变细*/

  • 字体样式

    font-style: italic;/*倾斜*/
font-style: normal;/*直立*/

  • font的复合属性

    /*font: font-style font-weught font-size/lin-height fpnt-family; 语法规范不能颠倒顺序*/ 
/*不需要的行可以省略，但是font-size font-*family不能省略*/
font: italic 700 16px/28px 'Microsoft yahei' ;/*将font属性写在一行，减少代码*/

    文本

  • 文本颜色

    color: #ff6600;/*十六进制改颜色,#开头，不需要记鼠标放在"#ff6600"*/
color: rgb(250,0,0);/*三基色表示颜色*/

  • 对齐文本

    text-align: center;/*相对于文本的盒子居中对齐*/
    属性值 作用
    left 左对齐（默认）
    right 右对齐
    center 居中对齐

  • 装饰文本

    text-decoration: line-through;/*x修饰文本的样式 加删除线*/
    属性值 作用
    none 默认，没有下划线，最常用
    underline 下划线，链接a自带下划线
    overline 上划线
    line-through 删除线

  • 首行缩进

    /*text-indent只会首行缩进，输入负值会向左缩进*/
text-indent: 20px;/*输入一个数值，px像素*/
text-indent: 2em;/**em表示当前文字的单位距离*/

  • 行间距

    line-height: 26px;/*行间距改变上下行的距离*/

    样式表

  • 内部样式表
    内部样式表是写在html页面内部，将css代码抽出，单独放在style标签中，控制整个html页面

    <style>
p {
   color: #160545;
   }
</style>

  • 行内样式表
    行内样式表写在标签里，仅仅对该标签进行修改，用于属性较少的情况，使用较少

    <p style="color: #165452;font: 54px sans-serif;">有点意思</p>/*stylel里放要修改的的属性*/

  • 外部样式表
    外部样式表，将css单独写在一个css文件里，再把css文件引入到html界面中，用于样式较多的情况，完全实现样式和结构分离，可控制多个页面，实际开发常用

1.新建一个css文件，将css代码写在文件里
2.在html页面中，使用标签引入到其中

<link rel="stylesheet" href="style.css">/*放在head标签下，css文件要和html在同一目录之下*/

Chrome调试工具

  • 打开调试工具
    打开Chrome浏览器，点击F12或者点击右键选择检查
  • 使用Chrome调试工具
  1. Ctrl+滚轮可以放大开发者代码大小
  2. 左边是html结构，右边是CSS样式
  3. 右边CSS样式可以直接改动数值和看属性值
  4. Ctrl+0恢复浏览器的大小
  5. 点击元素，发现没有样式引入，可能是类名或样式引入错误
  6. 如果样式前面出现黄色警告，则可能是属性书写错误
    注意：在浏览器上修改，必须将修改的内容复制到源码里，否则浏览器刷新，会恢复原貌
