简介 复合选择器、背景设置

Emmet语法

Emmet语法可以快速生成CSS语法和HTML语法，提高了代码书写

  • 快速生成html
    1.标签*n，即可快速生成n个标签
    2.父子级关系的，div>span,按tab
    3.兄弟级关系的，div+p,按tab
    4.带有类名，.nav,按tab，即可生成一个默认nav属性的div标签
    5.带有id，#nav,按tab，即可生成一个默认id为nav的div标签
    6.span.gray，按tab，即可生成一个gary属性的span标签
    7.span#gray，按tab，即可生成一个id为gary的span标签
    8..nav$*5,按tab，即可生成五个个默认nav属性的div标签，类的属性值从nav1-nav5

  • 快速生成CSS
    简写属性的首字母，按tab，即可快速补全代码

    复合选择器

    复合选择器是由两个或者两个以上的基础选择器，通过不同方式组合而成

    后代选择器

    <head>
<style>
  /*父标签和子标签用空格隔开*/
  ol li { /*选择ol中的li，而不是所有的li*/
      color: pink;
}
   .nav li{/*只会改变nav类的字体大小*/
    font-size: 25px;
  }
</style>
</head>
<body>
<ol>
  <li>123</li>/*会变色，不会变大小*/
</ol>
<ol class  = "nav">
  <li>4565</li>/*会变色，会变大小*/
</ol>
<ul>
  <li>13</li>/*颜色不会变化*/
</ul>
</body>

    元素1 元素2 {样式}，元素1和元素2可以是任意基础选择器
    元素2必须是元素1的后代，可以是多重后代

    子选择器

    子选择器只能选择作为某一元素的最近的一级元素

    <head>
<style>
  div > a {/*div和a之间用一个">"隔开*/
    color: #000;
  }
</style>
</head>
<body>
<div>
  <a href="#">亲儿子</a>/*会变颜色*/
  <p>
    儿子/*不会变颜色*/
    <a href="#">孙子</a>/*不会变颜色*/
  </p>
</div>
</body>

    元素1 >元素2 {样式}，元素1和元素2可以是任意基础选择器

    并集选择器

    并集选择器可以为多组标签定义相同的格式，用于集体声明

      div ,/*并集选择器常常竖着写*/
  p ,
  .nav {/*div,p和nav属性都改变了颜色,选择器可以是任意选择器*/
  color: red;
 }

    伪类选择器

  • 链接的伪类选择器

    a:link {
  color:aqua;
 }
    链接伪类选择器 作用
    a:link 选择所有未被访问的链接
    a:visited 选择所有已经被访问的链接
    a:hover 选择鼠标指针在其位置上的链接
    a:active 选择鼠标在其位置上摁下而未松开的链接

    按照:link - :visited -:hover -:active的顺序声明
    经常使用:hover选择器

  • focus伪类选择器
    focus伪类选择器用于获得光标的表单元素，一般input类表单元素才能获取.

input:focus{/*选择input表单中光标点击的元素*/
    background-color: aquamarine;
}
  • 一般元素的伪类选择器
    div:hover {/*用法和链接伪类选择器差不多*/
color: red;
}
div:hover span{/*当鼠标经过div时，div的子元素span颜色变化*/
color: red;
}

元素显示模式

  • 块元素
    1.独占一行
    2.可以设置宽度，高度，外边距和内边距都可以设置
    3.宽度默认是父类的的宽度
    4.可以放行内元素和块元素
    常见的块元素有h1-h6,p,div,ul,ol,li等.
  • 行内元素
    1.一行可以放多个元素
    2.宽和高直接设置是无效的
    3.默认宽度就是它本身内容的宽度
    4.不可以放块元素，只能放行内元素
    常见的行内元素有a,strong,b,em,i,del,s,ins,u,span
  • 行内块元素
    1.可以一行放多个元素，元素之间有缝隙
    2.默认宽度就是它本身内容的宽度
    3.高度，行高，外边距以及内边距都是可以设置的
    常见的有img,input, td
  • 元素显示模式转化
    简单来说将一个模式的元素转化为另一个模式的元素
    a {/*a标签是行内元素*/
 display: block;/*dispaly表示转换模式*/
 height: 153px;/*转换之后就可以设置宽度和高度*/
 width: 155px;
}
div {/*div标签是块元素*/
 display: inline;/*dispaly表示转换模式*/
 height: 153px;/*转换之后就不可以设置高度*/
}
span {/*span标签是行内元素*/
 display: inline-block;/*dispaly表示转换模式，转换为行内块元素*/
 height: 153px;/*转换之后就可以设置高度*/
}

    背景

  • 背景颜色
    background-color: #456415,pink,transparent;/*颜色可以直接写相应的单词，十六进制，transparent为透明*/
  • 背景图片
    background-image: url(images/R-C.png);/*images文件要和html文件在同一目录之下，文件目录要放在url()中，none默认没有背景图片*/
background-repeat: no-repeat;/*图片默认情况下视平铺的，no-repeat是平铺*//*盒子的大小不会改变*/
    • 平铺属性值
      属性值 作用
      repeat 平铺
      no-repeat 不平铺
      repeat-x 列平铺
      repeat-y 行平铺
    • 背景图片位置
      /*background-position: 方位名词;*/ 
/*仅仅写一个方位名词，则另一个默认居中*/
background-position: center center;/*center和top的顺序可以颠倒，位置没有影响*/
/*background-position: x y  仅写一个表示x*/
background-position: 20px 50px;/*20px设置x 50px设置y*/
/*两个可以混合使用，第一个是x，第二个是y*/
background-position: 20px center;/*20px设置x center设置y*/
    • 背景图片固定
      background-attachment: scroll;/*背景图像随内容滚动*/
background-attachment: fixed;/*背景图像固定*/
    • 背景复合写法
      /*background书写没有固定格式，一般background:color image repeat attachment position;*/
background: black url(images/R-C.png) no-repeat scroll center top ;
    • 背景颜色半透明
      background: rgba(0, 0, 0, 0.6) ;/*0.6设置透明度，0-1*/
