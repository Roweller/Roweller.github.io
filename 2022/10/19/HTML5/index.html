简介 HTML5的相关知识

标签的类型

根标签

<html>网页内容</html>

制作网页时，内容要都在根标签里面

html标签的子标签

<head></head>
<body></body>

放在html标签中

head标签的内容不会出现在网页中，设置网页的属性，看不见

body标签的内容会出现在网页中,能看见

  • head标签的子标签title
    <title>网页标题</title>
    搜索引擎点击后，超链接的页面标题

解析类型标签

<!doctype html>

告诉网页以html5解析网页，html5应用广泛，放在html标签之前

meta标签

meta标签放在head标签中

<meta charset ="utf-8">
<meta name = "keywords" content = "html,前端">
<meta http-equiv = "refresh" content ="3;url = https://www.baidu.com/">

meta标签放在head标签中
http-equiv 设置转到指定网页的时间，地址
name设置属性，content告诉搜索引擎网页的关键字

  • name不同属性的设置：
    标签 作用
    charset 网页字符集
    kewords 网页搜索的关键字
    discription 网页的描述
    http-equiv 设置转到的网页
    keywords 网页搜索的关键字
    discription 网页的描述 显示在搜索引擎界面

排版标签

  • 头部标签
    放在body标签中

    <header>头部</header>

    某一部分的头部部分

  • 主要主体标签
    放在body标签中

    <main>主体</main>

    网页的主体部分，一个网页只能有一个main标签

  • 底部标签
    放在body标签中

    <footer>底部</footer>

  • 导航标签
    放在body标签中

    <nav>导航</nav>

    便于查找网页内容

  • 边框标签
    放在body标签中

    <aside>导航</aside>

    放在网页的边栏部分

    表格标签

    <!--table表示表格，width和height限定表格的大小，border表示行距-->
<table align="center" width ="500" height ="250" border="1" >
  <tr><!-- tr表示每一行 -->
      <th>排名</th><!-- th表格头部 -->
      <th>名字</th>
  </tr>
  <tr>
      <td>123</td><!-- td表格主体 -->
      <td><a href="">x西游记</a> </td>
  </tr>
</table >
    属性名 属性值 作用
    align left、certer、right 规定表格对周围元素的对齐方式
    border 1 or “” 规定单元格是否有边框
    cellpadding 像素值 规定单元格与其内容之间的空白
    width 像素值或百分比 规定表格宽度

    文章的标签

  • 文章标签

    <article>导航</article>

    放在body标签中

  • 语义化标签

    标签 作用
    em 语气加重，斜体
    strong 强调，加粗
    blobkquote 引用标签

  • 分组标签

    <hgroup>
  <h1></h1>
  <h2></h2>
</hgroup>

    将有关的的标签放在hgroup中

  • 标题标签 

    <h1>****</h1>

    内容****放在标签中间，第二个标签加/与第一个标签进行区分，
    h1表示一级标签，h2则表示二级标签，h1-h6

  • 段落标签

    <p>$$$</p>

    与标题标签用法相同,表示一个段落

    标签的属性

  • 标签属性的分类
    标签属性分类转自独家记忆0408

  • 标签属性的使用

    属性 作用
    font-style 规定字体样子
    font-variant 规定字体异体
    font-weight 规定字体粗细
    font-size/line-height 规定字体尺寸和行高
    font-family 规定字体系列
    caption 定义被标题控件（比如按钮、下拉列表等）使用的字体
    icon 定义被图标标记使用的字体
    menu 定义被下拉列表使用的字体
    message-box 定义被对话框使用的字体
    small-caption caption 字体的小型版本
    status-bar 定义被窗口状态栏使用的字体 
    <font color = "yellow" size ='3'>内容</font>

    color设置“内容”的颜色，属性值yellow放在””中,属性之间用空格隔开

    块元素和行内元素

  • 介绍
    块元素：宏观展示，大多数都能放，用了会换行
    行内元素：对一行内的元素，进行添加修饰，不会换行
    行内元素一般可放在块元素中，而块元素很少放在行内元素中，用错了浏览器会自动修正

  • 分类
    块元素

    元素 标签
    行内元素 em,trong,p,a
    块元素 blockquote,h

列表

  • 无序列表
    ul表示列表标签
    li表示列表的项

    <ul>
  <li>结构</li>
  <li>表现</li>
</ul>

  • 有序列表
    ol表示列表标签
    li表示列表的项

    <ol>
  <li>结构</li>
  <li>表现<li>
</ol>

  • 定义列表
    dl表示定义列表
    dt表示要下定义的东西

</dl>

超链接

  • 介绍
    超链接路径分为相对路径和绝对路径
    打开同一目录下的网页，用相对路径
    打开外部网页，用绝对路径
  • 网页跳转
    <a href = "https://www12.baidu.com/" target = "_blank">超链接</a>
    href填网址，既可是绝对路径，也可是相对路径 target填是否打开新的网页，_blank代表打开，默认不打开
  • 超链接相互跳转
    <a id = "top" href = "#bottom" target = "_blank">超链接</a>
<a id = "bottom" href = "#top" target = "_blank">超链接</a>
    为超链接赋给一个id，href填入#+id，即可到达该网页指定超链接的位置
    href填入#，则默认返回该网页顶部，填入”javascrip:;”则不会跳转到任何界面

图片

  • 图片分类

    图片类型 特点 用途
    jpg 支持的颜色比较丰富，不支持透明效果，不支持动图 一般用于显示照片
    gif 支持颜色比较少，支持简单透明，支持动图 动图
    png 支持颜色丰富，支持复杂透明，不支持动图 网页
    base64 通过字符的形式引入图片 加载速度快

  • 用法

    <img src = "https://tse3-mm.cn.bing.net/th/id/OIP-C.Rq5h9rKa8uBoUxtGTCCVRQHaKd?w=130&h=184&c=7&r=0&o=5&dpr=1.3&pid=1.7" width = "800" alt = "风景">
<img src = "./img.gif" alt = "1">

    src填本地图片的路径，图片要和网页在同一目录下，也可填图片网址
    width设置图片的大小
    alt对图片进行命名，一般不会在网页上显示，而会便于搜素引擎查找

内联框架

  • 介绍
    引入别的网页在自己的网页中，放在一个指定大小的框架中
  • 用法
    <iframe src ="https://www.baidu.com" width = "800" height = "600" framebordor = "0"></iframe>
    src放网址，width和height设置大小，framebordor边框设置

音频、视频

音频

<audio contrals>
    <source src = "./scorce/ji.mp3" >
    <embed src = "./scorce/ji.mp3" type = "audio/mp3 " width = "400" height = "300">
</audio>

contrals是否允许用户控制播放
sourse用于windows9以上的浏览器
embed用于windows8以下的浏览器

    <embed src = "./scorce/ji.mp4" type = "audio/mp4 " width = "400" height = "300">
</video>

contrals是否允许用户控制播放
sourse用于windows9以上的浏览器
embed用于windows8以下的浏览器
